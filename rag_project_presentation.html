04 / 06 · CODE
Smart Ingestion
ingest.py — ตรวจจับไฟล์ใหม่ด้วย SHA-256 hash
# โหลด manifest ที่เคย ingest ไปแล้ว manifest = load_manifest() # {path: sha256_hash} # หาไฟล์ที่ยังไม่เคยเจอ หรือถูกแก้ไข new_files = scan_new_files(manifest) for path in new_files: chunks = splitter.split_documents( load_documents(path) ) vector_db.add_documents(chunks) manifest[str(path)] = sha256(path) save_manifest(manifest) # บันทึก manifest ใหม่
ingested_manifest.json vector_db/ my_documents/

ไฟล์ที่เคย ingest แล้วจะถูกข้ามโดยอัตโนมัติ — รันซ้ำได้เรื่อยๆ ปลอดภัย 100%
